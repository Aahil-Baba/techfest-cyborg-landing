# SYS_INIT // Techfest 29 Landing Page

A futuristic, interactive single-page application built for the **Techfest IIT Bombay Campus Ambassador Program**. 

Designed around the "Simulated Paradigm" theme, this landing page synthesizes premium UI/UX design with a cyberpunk aesthetic to effectively market Asia's largest science and technology festival. 

## Core Upgrades & Features

* **Terminal Boot Sequence:** A JavaScript-timed preloader that simulates an internal OS startup before revealing the main interface.
* **Human vs. Synthetic Toggle:** A custom light/dark mode switch that triggers a CSS glitch animation and swaps global UI variables to a high-contrast neon aesthetic.
* **Biometric Authentication:** A gamified "Hold-to-Verify" call-to-action button utilizing custom timing events and animated progress widths.
* **Glassmorphic UI Engine:** Premium translucent components utilizing CSS `backdrop-filter`, angular `clip-path` geometry, and neon bloom box-shadows.
* **Dynamic Media Modules:** Promotional cards featuring grayscale-to-color hover transitions to showcase core events like Robowars, Hackathons, and Technoholix.

## Tech Stack

* **Structure:** HTML5
* **Styling:** CSS3 (Custom CSS Properties, Grid/Flexbox Layouts, Transitions, Keyframe Animations)
* **Logic:** Vanilla JavaScript
